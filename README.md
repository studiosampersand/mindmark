# MindMark v0.000.001

Gratis, lokale mood- en habit tracker als installable PWA.

## Publiceren op GitHub Pages

1. Maak een nieuwe publieke repository.
2. Upload de volledige inhoud van deze map naar de root van de repository.
3. Open **Settings → Pages**.
4. Kies **Deploy from a branch**.
5. Selecteer `main` en `/ (root)`.
6. Bewaar en open de gepubliceerde URL.

## Paden

- `/` publieke SEO-homepage
- `/app/` stabiele lokale app
- `/beta-test/` aparte beta met eigen IndexedDB-opslag
- `/faq/`
- `/privacy-legal/`
- `/contact/`

## Belangrijk voor publicatie

- Vervang de placeholder GitHub Issues-URL in `contact/index.html`.
- Laat crisisinformatie, vertalingen en juridische teksten professioneel controleren.
- Controleer de naam MindMark formeel op domein-, handelsnaam- en merkrechten.
- Voeg een eigen `LICENSE` toe.
- Voor productie: maak PNG PWA-iconen als sommige platformen SVG-iconen niet aanvaarden.

## Privacy

De app gebruikt IndexedDB. Er is geen backend, account, analytics of externe runtimebibliotheek.
